##Guia para executar na sua maquina

Iniciar
Para executar o projeto é muito simples, basta seguir os passos abaixo.

Clone o repositório e abra o projeto descompactado e execute o seguinte comando no terminal:

Instalando

yarn install
//ou
npm install

iniciar

yarn start
//ou
npm start
